# Implementation = 피지컬로 승부하기
## 구현이란?
머릿 속에 있는 알고리즘을 소스코드로 바꾸는 과정 <br/><br/>

## 구현종류
### 완전탐색이란?
모든 경우의 수를 주저 없이 다 계산하는 해결 방법
- 반복문
- 재귀함수
- DFS/BFS 사용
- [주의] <br/>
    예외 케이스를 모두 확인해야 하는 경우가 많다! <br/><br/>

### 시뮬레이션이란?
문제에서 제시한 알고리즘을 한 단계씩 차례대로 직접 수행해야하는 문제 유형 <br/><br/>
- 완전 탐색과 비슷
